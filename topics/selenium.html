🎭 Selenium WebDriver คืออะไร?

Selenium WebDriver เป็น open-source tool สำหรับ automate การทดสอบ web application โดยควบคุม browser จริงๆ (Chrome, Firefox, Safari, Edge)

✨ ทำไมต้อง Selenium?

  • 🌐 รองรับทุก browser หลัก
  • 🔤 รองรับหลายภาษา (Python, Java, JavaScript, C#, Ruby)
  • 💰 ฟรี (Open Source)
  • 🏢 มาตรฐานอุตสาหกรรม
  • 🔧 ยืดหยุ่น ปรับแต่งได้

🏗️ สถาปัตยกรรม Selenium

Test Script (Python/Java/JS)
         ↓
WebDriver (Selenium Library)
         ↓
Browser Driver (ChromeDriver/GeckoDriver)
         ↓
    Browser

⚙️ การติดตั้ง (Python)

# 1. ติดตั้ง Selenium Library
pip install selenium

# 2. ดาวน์โหลด ChromeDriver (ต้องตรงกับ Chrome version)
# https://chromedriver.chromium.org/downloads

# 3. วาง chromedriver ใน PATH หรือระบุ path ตรงๆ

🚀 Hello World - เปิดเว็บด้วย Selenium

from selenium import webdriver
from selenium.webdriver.common.by import By

# เปิด Chrome Browser
driver = webdriver.Chrome()

# เปิดเว็บไซต์
driver.get("https://www.google.com")

# หา search box และพิมพ์ค้นหา
search_box = driver.find_element(By.NAME, "q")
search_box.send_keys("Selenium WebDriver")
search_box.submit()

# รอ 5 วินาทีแล้วปิด
import time
time.sleep(5)
driver.quit()

🎯 Locators - หา Element บนหน้าเว็บ

การหา element เป็นสิ่งสำคัญใน Selenium มีหลายวิธี:

Locator คำอธิบาย ตัวอย่าง ความเร็ว
ID หาจาก id attribute (ควรใช้) By.ID, "username" ⭐⭐⭐⭐⭐
NAME หาจาก name attribute By.NAME, "email" ⭐⭐⭐⭐
CLASS_NAME หาจาก class attribute By.CLASS_NAME, "btn-primary" ⭐⭐⭐
TAG_NAME หาจาก tag name By.TAG_NAME, "button" ⭐⭐
LINK_TEXT หาลิงก์จากข้อความเต็ม By.LINK_TEXT, "Click Here" ⭐⭐⭐
PARTIAL_LINK_TEXT หาลิงก์จากข้อความบางส่วน By.PARTIAL_LINK_TEXT, "Click" ⭐⭐⭐
CSS_SELECTOR ใช้ CSS selector (ยืดหยุ่น) By.CSS_SELECTOR, "#login .btn" ⭐⭐⭐⭐
XPATH ใช้ XPath (ทรงพลัง) By.XPATH, "//input[@id='user']" ⭐⭐

💡 Best Practice การเลือก Locator

ลำดับความสำคัญ: ID > NAME > CSS_SELECTOR > XPATH > CLASS_NAME > TAG_NAME

🖱️ Actions - โต้ตอบกับ Element

พื้นฐานที่ต้องรู้

from selenium.webdriver.common.by import By

# หา element
button = driver.find_element(By.ID, "submit-btn")

# คลิก
button.click()

# พิมพ์ข้อความ (input fields)
input_field = driver.find_element(By.NAME, "username")
input_field.send_keys("myusername")

# ล้างข้อความ
input_field.clear()

# ดึงข้อความจาก element
text = button.text

# ดึง attribute value
link_url = driver.find_element(By.TAG_NAME, "a").get_attribute("href")

# ตรวจสอบว่า element แสดงอยู่ไหม
is_visible = button.is_displayed()

# ตรวจสอบว่า element ใช้งานได้ไหม
is_enabled = button.is_enabled()

🖱️ Advanced Actions (ActionChains)

from selenium.webdriver.common.action_chains import ActionChains

actions = ActionChains(driver)

# Hover (Mouse Over)
element = driver.find_element(By.ID, "menu")
actions.move_to_element(element).perform()

# Right Click
actions.context_click(element).perform()

# Double Click
actions.double_click(element).perform()

# Drag and Drop
source = driver.find_element(By.ID, "draggable")
target = driver.find_element(By.ID, "droppable")
actions.drag_and_drop(source, target).perform()

# คลิกและถือ (Click and Hold)
actions.click_and_hold(element).move_by_offset(100, 0).release().perform()

⏱️ Waits - การรอ Element

เว็บสมัยนี้โหลดแบบ dynamic ต้องรอ element ปรากฏก่อนคลิก

❌ อย่าใช้ time.sleep()!

# ❌ ไม่แนะนำ - รอแบบ fixed time
time.sleep(5)  # รอ 5 วินาทีเสมอ ไม่ว่าเว็บจะโหลดเร็วหรือช้า

✅ ใช้ Explicit Wait (แนะนำ)

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# รอสูงสุด 10 วินาที จนกว่า element จะคลิกได้
wait = WebDriverWait(driver, 10)

# รอจนกว่าจะคลิกได้
button = wait.until(EC.element_to_be_clickable((By.ID, "submit")))
button.click()

# รอจนกว่าจะมองเห็น
message = wait.until(EC.visibility_of_element_located((By.CLASS_NAME, "success")))

# รอจนกว่าจะปรากฏ
element = wait.until(EC.presence_of_element_located((By.TAG_NAME, "div")))

📋 Expected Conditions ที่ใช้บ่อย

Conditionคำอธิบาย
presence_of_element_locatedElement มีอยู่ใน DOM
visibility_of_element_locatedElement มองเห็นได้ (displayed)
element_to_be_clickableElement คลิกได้ (visible + enabled)
text_to_be_present_in_elementElement มีข้อความที่ต้องการ
title_containsTitle ของหน้าเปลี่ยน

💼 ตัวอย่าง: Login Test Case

import pytest
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

class TestLogin:
    
    @pytest.fixture
    def driver(self):
        driver = webdriver.Chrome()
        driver.get("https://example.com/login")
        yield driver
        driver.quit()
    
    def test_successful_login(self, driver):
        """ทดสอบ login สำเร็จ"""
        wait = WebDriverWait(driver, 10)
        
        # กรอก username
        username = driver.find_element(By.ID, "username")
        username.send_keys("testuser")
        
        # กรอก password
        password = driver.find_element(By.ID, "password")
        password.send_keys("Test@123")
        
        # คลิก login
        login_btn = driver.find_element(By.ID, "login-btn")
        login_btn.click()
        
        # รอและตรวจสอบว่า login สำเร็จ
        welcome_msg = wait.until(
            EC.visibility_of_element_located((By.CLASS_NAME, "welcome-message"))
        )
        assert "Welcome" in welcome_msg.text
        assert "dashboard" in driver.current_url
    
    def test_invalid_login_shows_error(self, driver):
        """ทดสอบ login ผิดพลาดแสดง error"""
        driver.find_element(By.ID, "username").send_keys("wronguser")
        driver.find_element(By.ID, "password").send_keys("wrongpass")
        driver.find_element(By.ID, "login-btn").click()
        
        error = WebDriverWait(driver, 10).until(
            EC.visibility_of_element_located((By.CLASS_NAME, "error-message"))
        )
        assert "Invalid credentials" in error.text
