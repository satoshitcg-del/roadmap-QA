🔍 SQL Testing

Test Case Examples

-- Test 1: Verify SELECT returns correct data
SELECT * FROM users WHERE id = 123;
-- Expected: 1 row with correct user data

-- Test 2: Verify INSERT works
INSERT INTO users (name, email) VALUES ('Test', 'test@email.com');
-- Expected: 1 row affected, auto-increment ID assigned

-- Test 3: Verify UPDATE
UPDATE users SET status = 'active' WHERE id = 123;
-- Expected: Only row 123 updated, status changed

-- Test 4: Verify DELETE with constraints
DELETE FROM orders WHERE id = 999;
-- Expected: Row deleted or FK constraint error if referenced

-- Test 5: Verify JOIN returns correct data
SELECT u.name, o.total
FROM users u
JOIN orders o ON u.id = o.user_id
WHERE u.id = 123;
-- Expected: All orders for user 123

🧪 Testing with Python

import pytest
import mysql.connector

class TestDatabase:
    
    def setup_method(self):
        self.conn = mysql.connector.connect(
            host='localhost',
            user='test_user',
            password='password',
            database='test_db'
        )
        self.cursor = self.conn.cursor()
    
    def teardown_method(self):
        self.cursor.close()
        self.conn.close()
    
    def test_user_insert(self):
        """ทดสอบ INSERT user"""
        query = "INSERT INTO users (name, email) VALUES (%s, %s)"
        self.cursor.execute(query, ('John Doe', 'john@email.com'))
        self.conn.commit()
        
        assert self.cursor.rowcount == 1
        assert self.cursor.lastrowid > 0
    
    def test_user_select(self):
        """ทดสอบ SELECT user"""
        self.cursor.execute("SELECT * FROM users WHERE id = 1")
        result = self.cursor.fetchone()
        
        assert result is not None
        assert result[1] == 'John Doe'  # name column
    
    def test_foreign_key_constraint(self):
        """ทดสอบ Foreign Key constraint"""
        # ลอง insert order ด้วย user_id ที่ไม่มีอยู่
        query = "INSERT INTO orders (user_id, total) VALUES (999, 100)"
        
        with pytest.raises(mysql.connector.IntegrityError):
            self.cursor.execute(query)
            self.conn.commit()
    
    def test_data_count(self):
        """ทดสอบ row count"""
        self.cursor.execute("SELECT COUNT(*) FROM users")
        count = self.cursor.fetchone()[0]
        
        assert count >= 0  # หรือตรวจสอบค่าเฉพาะ

✅ Database Testing Checklist

Schema Validation

  • ✓ Table names correct
  • ✓ Column names and data types
  • ✓ Column lengths and constraints
  • ✓ Indexes defined correctly

Data Validation

  • ✓ No orphaned records (FK integrity)
  • ✓ Default values set correctly
  • ✓ No duplicate data where unique required
  • ✓ Date/time formats consistent

CRUD Operations

  • ✓ Create (INSERT) with valid data
  • ✓ Read (SELECT) returns correct results
  • ✓ Update (UPDATE) modifies only intended rows
  • ✓ Delete (DELETE) handles constraints properly
