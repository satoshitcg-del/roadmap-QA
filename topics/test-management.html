🚂 TestRail - ตัวอย่าง Tool

Key Features

  • Test Cases - จัดการ test cases เป็นหมวดหมู่ (Suites, Sections)
  • Test Runs - สร้าง test run สำหรับแต่ละ release/sprint
  • Milestones - จัดเก็บผลการทดสอบตาม milestone
  • Reports> - Progress, Comparison, Coverage reports
  • Integrations> - Jira, GitHub, CI/CD tools

TestRail Workflow

  1. Create Test Suite (e.g., "Login Feature")
  2. Add Test Cases (with steps, expected results)
  3. Create Test Run (select test cases for this release)
  4. Execute Tests (mark Pass/Fail/Block)
  5. Push bugs to Jira (integration)
  6. Generate Report

🎯 Jira + Zephyr/Xray

ถ้าทีมใช้ Jira อยู่แล้ว การใช้ plugin จะสะดวกกว่า

Zephyr Scale (TM4J)

  • Test cases อยู่ใน Jira issues
  • Link test cases กับ stories/requirements
  • Traceability matrix
  • Reuse test cases across projects

Xray for Jira

  • รองรับ BDD (Cucumber, Gherkin)
  • Integration กับ automation frameworks
  • Test Execution เป็น Jira issue type
