Bug คืออะไร?

Bug หรือ Defect คือความแตกต่างระหว่างผลลัพธ์จริงกับที่คาดหวัง

ส่วนประกอบ Bug Report ที่ดี

  • Title/Summary: สั้น กระชับ บอกปัญหาและผลกระทบ
  • Description: รายละเอียดปัญหา
  • Steps to Reproduce: ขั้นตอนที่ทำให้เกิดบั๊ก
  • Expected Result: ควรเป็นอย่างไร
  • Actual Result: เป็นจริงอย่างไร
  • Environment: Browser, OS, Version
  • Screenshots/Logs: หลักฐานประกอบ
