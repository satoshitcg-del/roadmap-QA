🛡️ OWASP Top 10 (2021)

OWASP (Open Web Application Security Project) จัดอันดับความเสี่ยงด้านความปลอดภัยที่พบบ่อยที่สุด

Rank Vulnerability คำอธิบาย
A01 Broken Access Control การควบคุมสิทธิ์มีปัญหา
A02 Cryptographic Failures การเข้ารหัสผิดพลาด
A03 Injection SQL, NoSQL, OS command injection
A04 Insecure Design ออกแบบระบบไม่ปลอดภัย
A05 Security Misconfiguration ตั้งค่าความปลอดภัยผิด
A06 Vulnerable Components ใช้ component/library ที่มีช่องโหว่
A07 Auth Failures การยืนยันตัวตนผิดพลาด
A08 Data Integrity Failures ความสมบูรณ์ข้อมูลผิดพลาด
A09 Security Logging Failures ไม่มีการบันทึกเหตุการณ์ความปลอดภัย
A10 SSRF Server-Side Request Forgery

💉 SQL Injection

อะไรคือ SQL Injection?

การโจมตีโดยแทรก SQL code เข้าไปใน input fields เพื่อ manipulate database

❌ Vulnerable Code

# ไม่ปลอดภัย! ต่อ string ตรงๆ
query = "SELECT * FROM users WHERE username = '" + username + "'"
cursor.execute(query)

✅ Secure Code (Parameterized Query)

# ปลอดภัย - ใช้ parameterized query
query = "SELECT * FROM users WHERE username = %s"
cursor.execute(query, (username,))

🔴 Cross-Site Scripting (XSS)

ประเภทของ XSS

  • Stored XSS - Script เก็บไว้ใน database
  • Reflected XSS - Script อยู่ใน URL/params
  • DOM-based XSS - เกิดจาก client-side JavaScript
